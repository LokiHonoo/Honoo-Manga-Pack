## 简介

漫画和图包的打包辅助工具，WPF 窗体类型。

支持 ZIP/RAR/7Z/PDF 解压缩或重新打包到无压缩 ZIP 文件。

## 引用组件

[SharpCompress](https://github.com/adamhathcock/sharpcompress)

[iText 8](https://itextpdf.com/)

[HandyControl](https://github.com/HandyOrg/HandyControl)
